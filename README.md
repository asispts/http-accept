# HTTP Accept
PHP Parser for dealing with HTTP `Accept`, `Accept-Language`, `Accept-Encoding`, and `Accept-Charset` headers

## Installation
Use [composer](https://getcomposer.org/) to install the library.
```bash
composer require xynha/http-accept
```
